2 Player Pinball-Machine

I am currently working on a 2 player pinball machine. The machine is designed to be played by two players at the same time. The machine is controlled by an Arduino and uses a Raspberry Pi to display the score.

LED Smart-Clock

I have built a smart clock that can display the time, date, and temperature. The clock is controlled by an Arduino and uses a real-time clock module to keep track of the time. The clock is also equipped with an alarm function.

Midi Volume Sliders

I have built a set of MIDI volume sliders that can be used to control the volume of a digital audio workstation. The sliders are controlled by an Arduino and use a MIDI interface to communicate with the computer.
